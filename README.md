## zephyr-node

#### Dependencies

1. [Install Docker](https://docs.docker.com/get-docker/)
2. [Install golang 1.16](https://golang.org/doc/install)
3. [Install Protobuf Compiler](https://grpc.io/docs/protoc-installation/)


#### Build Node

```shell
$ make build
```

#### Run Node

```shell
$ ./zephyr-node
```
